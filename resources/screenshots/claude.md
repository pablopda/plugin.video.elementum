# screenshots

## Purpose

Marketing and documentation screenshots showing the plugin interface.

## Contents

- **home.png** - Main plugin home screen
- **movies.png** - Movie browsing interface
- **webui.png** - Web management interface

## Usage

Referenced in `addon.xml` for Kodi addon browser:
```xml
<screenshot>resources/screenshots/home.png</screenshot>
<screenshot>resources/screenshots/movies.png</screenshot>
<screenshot>resources/screenshots/webui.png</screenshot>
```

## Specifications

- Format: PNG
- Resolution: 1280x720 or higher recommended
- Content: Clean UI without personal data
- Purpose: Show addon features to potential users

## Notes

- Keep screenshots updated with major UI changes
- Use default Kodi skin for consistency
- Avoid showing copyrighted content
